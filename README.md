# Islamic Online Library (Advanced E-commerce Platform)

## Overview
A premium, high-performance e-commerce platform inspired by Wafilife, built with a modern MERN stack.

## Tech Stack
- **Frontend**: React (Vite), TypeScript, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Tools**: Redux Toolkit, React Query, React Router DOM

## Features
- Advanced Search
- User Authentication & Profiles
- Admin Dashboard
- Payment Gateway Integration
- Responsive Premium Design
